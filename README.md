## Setup

1. Download project and run web server:

  ```bash
  # Clone the project
  git clone https://github.com/clemans/net.clemans/Project/WDIDTP.git

  # Install project dependencies
  npm install

  # Run the web server
  npm run start
  ```

2. Browse to: <http://localhost:3000>
