## Setup

1. Download project and run web server:

  `git clone https://github.com/clemans/net.clemans/Project/WDIDTP.git`

2. Install project dependencies:

 `npm install`

3.  Run the web server:
  
 ` npm run start`

4. Browse to: <http://localhost:3000>
